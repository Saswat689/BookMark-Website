## Installlation

- Run `npm start` to install tailwind in your directory.
- Run `npm run tailwind:build` to build up the css

## Features

- Reveal on scroll animations (wow.js && animate.css)
- Completely responsive
- Only HTML design (Tailwind css)
- Navigation
- Forms
